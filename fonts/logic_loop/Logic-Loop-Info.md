# Logic Loop
Logic Loop is a bold 13pt pixel art font designed with tall lettering and some sharp angles giving it a slight futuristic vibe.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 13px
- Scales: 13, 26, 39, 52, etc
- Ascender Max: 6px
- Descender Max: 5px

## Kerning
None
